## About

A CSCW (Computer-Supported Cooperative Work) project exploring responsible design of affective / collaborative interaction technology, building on course themes:

- Bush (1945) — *As We May Think* (memex, associative trails)
- Weiser (1991) — *The Computer for the 21st Century* (ubiquitous computing)
- ÖAW — *Narratives of Digital Ethics* (situated digital ethics)
- HCI/AI "superpowers" framing (Lecture 1)
